# 📦 FitBox3D - Cálculo de Containers

Sistema web desenvolvido para auxiliar empresas de logística no cálculo do volume ocupado por cargas dentro de containers.

## 🎯 Objetivo

Determinar a quantidade de containers necessários para transportar mercadorias, com base nas dimensões e volume da carga.

## 🚚 Funcionalidades

- Cálculo de volume de caixas
- Estimativa de ocupação de containers
- Interface interativa para entrada de dados
- Simulação de transporte logístico

## 💻 Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## 📦 Aplicação prática

Este sistema pode ser utilizado por empresas de logística para:

- Planejamento de transporte
- Redução de custos operacionais
- Melhor aproveitamento de espaço

## ▶️ Como executar

1. Baixe o projeto
2. Abra o arquivo `index.html` no navegador
